## How It Works

### 1. Authentication Flow
```
User visits form.html
    ↓
form.js loads and runs checkAuthentication()
    ↓
Checks sessionStorage.getItem('sessionAuthN')
    ↓
If false/null → Redirects to auth.html
If true → Loads form normally
```

**Authentication Setup (auth.html):**
- Login credentials: username (any), password: `una2026`
- On successful login: sets `sessionStorage.setItem('sessionAuthN', true)`
- Redirects to home page

### 2. Form Data Collection
```
User fills form with:
- Title (required)
- Author (required)
- Date (required - auto-set to today)
- Image (optional - converted to base64)
- Description (required)
- Links (optional - multiple entries)
  - URL (optional)
  - Description (optional)
```

### 3. Data Storage Structure
```javascript
// localStorage key: 'contentItems'
// Value: JSON array of content objects

[
  {
    id: "content_1718449800000_abc123def",
    title: "My Article",
    author: "John Doe",
    date: "2026-06-15",
    image: "data:image/png;base64,...",
    description: "Full description text...",
    links: [
      {
        url: "https://example.com",
        description: "Example site"
      },
      {
        url: "https://another.com",
        description: null
      }
    ],
    createdAt: "2026-06-15T14:30:00.000Z"
  },
  // More items...
]
```

### 4. Form Submission Flow
```
User submits form
    ↓
Form validation (HTML5 + Bootstrap)
    ↓
Collect all form data
    ↓
Convert image to base64 (if present)
    ↓
Generate unique ID + timestamp
    ↓
Save to localStorage['contentItems'] array
    ↓
Show success message
    ↓
Reset form to initial state
```

### 5. Content Integration (content.html)
To load content items on content.html:
```javascript
const contentItems = JSON.parse(localStorage.getItem('contentItems')) || [];

contentItems.forEach(item => {
  // Create card with item.title, item.author, item.image, etc.
  // Display links as item.links array
});
```

## Key JavaScript Functions (form.js)

### `checkAuthentication()`
- Verifies user is logged in
- Redirects to auth.html if not authenticated

### `initializeForm()`
- Sets date to today
- Attaches event listeners
- Initializes form validation

### `addLinkInput()`
- Dynamically adds URL + description fields
- Includes remove button for each link

### `collectFormData()`
- Gathers all form inputs
- Converts image to base64
- Generates unique ID and timestamp

### `collectLinks()`
- Extracts all link entries
- Returns array of {url, description} objects

### `saveToLocalStorage(formData)`
- Retrieves existing contentItems array
- Adds new item
- Saves updated array back to localStorage

### `resetForm()`
- Clears all inputs
- Resets date to today
- Restores single default link input

### `showSuccessMessage()`
- Displays Bootstrap alert
- Auto-dismisses after 5 seconds

### `generateUniqueId()`
- Creates unique identifier for each content item
- Format: `content_[timestamp]_[random]`
