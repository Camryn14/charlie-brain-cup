# AGENTS.md

## Project

Course Catalog System

## Development Standards

This project follows the Google JavaScript Style Guide.

## Libraries

- Bootstrap 5
- Google Fonts (Roboto)

## Coding Standards

- Use semantic HTML5 elements.
- Keep JavaScript organized into functions.
- Use meaningful variable and function names.
- Keep code readable and properly indented.
- Use Bootstrap components for layout and styling.
- Use DOM manipulation to update page content.

## Features

- Display course cards
- Add new courses
- Edit existing courses using contenteditable="true"
- Delete courses
